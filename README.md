# simple_linear_search

Python program to carry out a linear search on a list data structure

## Description

This Python program is used to demonstrate a simple linear search on a list data structure

Python exercise: 

Write a Python program to carry out a linear search on a list data structure

Instruction: 
Start from the leftmost element of the list and one by one compare x with each element of the list  <br>
If x matches with an element, return True                                                           <br>
If x doesnâ€™t match with any of the elements, return False                                           <br>

## Getting Started

### Dependencies

This python project does not have any external libraries. However, starting this program will only require a Python 3 version.

### Installing

The latest version of python can be installed on the website: https://www.python.org/downloads/.
